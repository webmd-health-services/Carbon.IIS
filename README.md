The "Carbon.IIS" module...

# System Requirements

* Windows PowerShell 5.1 and .NET 4.6.1+
* PowerShell Core 6+

# Installing

To install globally:

```powershell
Install-Module -Name 'Carbon.IIS'
Import-Module -Name 'Carbon.IIS'
```

To install privately:

```powershell
Save-Module -Name 'Carbon.IIS' -Path '.'
Import-Module -Name '.\Carbon.IIS'
```
